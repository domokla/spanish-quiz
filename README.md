# Spanish Quiz

A simple private web app to practice Spanish vocabulary. 
Uses a curated CSV file (Hungarian → Spanish) for family and school learning.

## Features
- 20 random words per quiz session
- Instant feedback on answers
- Pronounce Spanish words (es-ES)
- Simple static HTML app
- Easy to host on GitHub Pages

## CSV Format
magyar,spanyol
alma,manzana
kutya,perro

## Getting Started
1. Clone the repo
2. Open index.html
3. Make sure your CSV is in the correct format
4. Start practicing!
